#  ContasCorrentes_io

Este projeto simula o funcionamento básico de um **sistema bancário**, permitindo o **gerenciamento de contas correntes**, **movimentações financeiras** e **atualização de saldos**.

##  Objetivo
O sistema foi desenvolvido com o intuito de praticar **manipulação de arquivos**, **estruturas de repetição**, **tratamento de dados** e **instancia de classes em Java**.

##  Estrutura do Projeto
O projeto é dividido em várias classes que representam cada parte do sistema:

- **MenuPrincipal.java** → Gerencia as opções do sistema e chama os métodos principais.  
- **ContasCorrentes.java** → Armazena os dados das contas.  
- **MovimentoContas.java** → Registra depósitos, saques e outras movimentações.  
- **ContasAtualizados.java** → Mostra o saldo final das contas após as operações.  
- **ClasseConsulta.java / ClasseMetodos.java** → Executam operações de leitura, gravação e atualização de arquivos.

##  Funcionalidades
- Cadastro de contas correntes.  
- Registro de movimentações (crédito e débito).  
- Atualização automática dos saldos.  
- Exibição das contas e seus valores atualizados.  

##  Tecnologias Utilizadas
- Linguagem: **Java**
- IDE recomendada: **NetBeans**
- Entrada/Saída de dados: `JOptionPane` e manipulação de arquivos `.txt`

##  Exemplo de execução
O usuário pode:
1. Cadastrar novas contas.  
2. Realizar depósitos e saques.  
3. Consultar o saldo final das contas após as atualizações.
